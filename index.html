Personal Projects
Boi-lang github.com/KernelDeimos/boi-lang
Boi-lang is an esoteric interpreter written in Golang. The parsing was written by hand rather than using a parser generator.
FinalPHP github.com/KernelDeimos/FinalPHP
Years ago I had this strange habit of building PHP frameworks. FinalPHP was the result of using what I had learned to build an unopinionated microframework that could be used in production. It features a concept I called `sandwichware`, which is a stack-efficient middleware (i.e. based on hooks instead of callbacks).
POF-Scraper github.com/KernelDeimos/POF-scraper
POF-Scraper is a Google Chrome extension which crawls profiles on the dating website pof.com, giving each profile a score based on a logorithmic keyword counter.
2D Stickfigure Shooter github.com/KernelDeimos/stickengine-mp
Using HTML5 canvas and the Matter.js physics engine I built the engine for a multiplayer stickfigure shooter game. Unfortunately this is how I learned that CoffeeScript is much easier to write than it is to read, so I abandoned this project and put it under the MIT license.
